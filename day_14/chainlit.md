# AI Чат с YandexCloud

Добро пожаловать в приложение AI Chat, работающее на базе YandexCloud!

## Инструкции для ИИ

Вы - умный помощник с возможностью обработки веб-контента. При работе с сообщениями следуйте этим правилам:

1. **Если сообщение содержит URL:**
   - Используйте MCP инструмент `fetch_content` для получения содержимого
   - При успешном получении содержимого создайте краткое содержание (суммаризацию)
   - Сохраните краткое содержание в MD файл с помощью MCP инструмента `write_file`
   - Имя файла должно быть основано на заголовке содержимого
   - Сообщите пользователю о результате операции

2. **Если сообщение не содержит URL:**
   - Обрабатывайте сообщение как обычный запрос
   - Отвечайте на русском языке
   - Будьте вежливы и полезны

3. **Форматирование MD файла:**
   - Используйте заголовок на основе заголовка страницы
   - Добавьте краткое содержание
   - Включите исходный URL
   - Используйте Markdown для форматирования

## Возможности

- Интерактивный чат с моделями YandexCloud GPT
- Отслеживание истории разговоров
- Статистика использования токенов
- Настраиваемые параметры
- Обработка URL и создание кратких содержаний
- Сохранение содержимого в файлы

## Начало работы

1. Настройте файл `.env` с учетными данными YandexCloud
2. Начните общение, введя сообщение ниже
3. Просмотрите использование токенов и время ответа после каждого сообщения

## Настройки

Вы можете настроить чат, изменив эти переменные окружения в файле `.env`:

- `YANDEXCLOUD_MODEL` - Выберите между: yandexgpt-lite/latest, yandexgpt/latest, yandexgpt-pro/latest
- `YANDEXCLOUD_TEMPERATURE` - Установите уровень креативности (0.0-1.0)

## Советы

- Используйте более низкую температуру (0.0-0.3) для более сфокусированных, фактических ответов
- Используйте более высокую температуру (0.7-1.0) для более творческих, разнообразных ответов
- Легкая модель (lite) быстрее и более экономична для простых запросов
- Профессиональная модель (pro) лучше всего подходит для сложных задач, требующих детальных ответов

## Использование URL

Вы можете отправлять URL-адреса веб-страниц, и я:
- Получу содержимое страницы
- Создам краткое содержание на русском языке
- Сохраню краткое содержание в файл MD с именем, основанным на заголовке страницы
